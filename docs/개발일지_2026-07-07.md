- **사이트**: 생성형 AI 기반 문서·인사행정 실무 자동화 (university.dreamitbiz.com)

## 1. 긴급 버그 수정 (재강의 필수)

| 항목 | 내용 |
|------|------|
| **라이브 로그인 전면 복구** | 배포본 Supabase anon key가 `placeholder-key`로 나가 로그인·회원가입·커뮤니티가 전부 작동 불가였음 → `supabase.ts` fallback을 실제 URL/anon key로 하드코딩(형제 사이트와 통일). REST 200·auth 엔드포인트 검증 완료 |

## 2. 실습 콘텐츠 대폭 확충 (8시간×2일 분량 확보)

재강의 시 진도 빠른/느린 학습자 모두 실습할 수 있도록 세 형식으로 강화.

### (1) 난이도별 실습 프롬프트 뱅크 (`practice-bank.ts`)
- 문서행정 5주제 + 인사행정 7주제 = **12개 주제, 주제당 ~10종**(🟢기초·🟡응용·🔴심화)
- 안전 설계: 프롬프트를 일반 문자열로 관리 → 펜스 이스케이프 불필요, **CopyBlock 복사 버튼 자동 적용**
- 공문서 작성에는 문서 유형별 프롬프트(기안문·통보문·협조문·회신문) 별도 포함

### (2) 완성형 실전 예시 (`WORKED_EXAMPLES`)
- 빈칸 없이 바로 실행되는 완성 프롬프트 + **예상 결과물** 설명
- 문서행정 6주제 + 인사행정 7주제, 주제당 2종 (강의 중 라이브 시연용)

### (3) 단계별 실습 워크시트 (`WORKSHEETS`)
- 한 주제를 A→Z로 완성하는 5단계 가이드(단계별 프롬프트 + 확인 포인트)
- 공문 1건 완성 / Excel 분석 완성 / 행사 문서 세트 / 채용 A→Z / 근태 분석 / 인사 임용 등

### (4) 복사 기능 (`CopyBlock.tsx`)
- 학습자료의 모든 프롬프트 블록에 복사 버튼 신설(기존엔 복사 불가)
